# HH IDE

React + Redux + Ant Design application.

## Tech Stack

- React 18 + TypeScript
- Redux Toolkit
- Ant Design
- Webpack 5

## Development

```bash
npm install
npm start          # http://localhost:3002
npm run build      # production build
```

## Usage

```typescript
import { useAppDispatch, useAppSelector } from '@/store/hooks';
```
