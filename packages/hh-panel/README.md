# HH Panel

React panel component library.

## Installation

```bash
pnpm add @huahuo/hh-panel
```

## Usage

```typescript
import { Panel } from '@huahuo/hh-panel';

function App() {
  return (
    <Panel title="My Panel" width={300}>
      <p>Panel content</p>
    </Panel>
  );
}
```

## Development

```bash
pnpm build   # Build the library
pnpm watch   # Watch mode
```
